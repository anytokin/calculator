## Installation

```sh
pip install numpy
pip install calculator
```

## Usage

With use of this tool you can:

-  add numbers
-  subtract numbers
-  multiply numbers
-  divide numbers
-  root numbers

```python
>>> calc = Calculator()
>>> calc.add(0.1)
0.1
>>> calc.add(0.2)
0.3
>>> calc.subtract(10.3)
-10.0
>>> calc.divide(3)
-3.333333333333
>>> calc.multiply(3)
-10.0
>>> calc.root(1/3)
-1000.0
>>> calc.reset()
0.0
```
